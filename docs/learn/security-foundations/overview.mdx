This section includes conceptual guides on application security essentials. These fundamental concepts can help strengthen your organization's security posture and can be a helpful reference when educating teams on security principles.

## Featured Articles[​](#featured-articles "Direct link to Featured Articles")

* [Static Application Security Testing (SAST)](/docs/learn/security-foundations/sast/overview): Identify vulnerabilities in applications before deployment with tools designed to analyze source code without executing it.

* [Supply Chain Security (SCA)](/docs/learn/security-foundations/supply-chain-security): Understand vulnerable dependencies that your source code relies upon.

* [Secure Development Workflows](/docs/learn/security-foundations/security-testing-workflow): Plan integration points for security testing into regular development workflows.
